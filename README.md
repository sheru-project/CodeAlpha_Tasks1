# 📱 Mini Social Media Platform

A full-stack mini social media web application where users can create profiles, share posts, interact through comments, and build connections via likes and follows.

This project demonstrates core concepts of modern web development including authentication, database relationships, and dynamic UI updates.

---

## 🚀 Features

- 👤 User Registration & Profiles
- 📝 Create, Edit, Delete Posts
- 💬 Comment on Posts
- ❤️ Like System
- ➕ Follow / Unfollow Users
- 📰 Dynamic Feed (Posts from followed users)
- 📱 Responsive UI Design

---

## 🛠️ Tech Stack

### Frontend:
- HTML5
- CSS3
- JavaScript (Vanilla JS)

### Backend:
- Node.js + Express.js  
  OR  
- Django (Python)

### Database:
- MongoDB (for Node.js)  
  OR  
- SQLite / PostgreSQL (for Django)
